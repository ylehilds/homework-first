Homework First

A homework tool designed for parents to help their children to prioritize homework before recreational computer screen time.

Learn More

The Vision

To help kids prioritize their screen time and not spend too much watching videos or playing games. This tool aims to automate the reminder/enforce them to focus on school first, before computer play time.

Canvas

A canvas integration that will use parent canvas account to look what are the upcoming homework tasks that the kids need to do.

Homework Configuration

Parents will be able to configure how many homework assignments will need to be done before unlocking their child's computer for play time. This will be a computer application that will remain locked until a goal is met.

Reward

Once the child hits the target on how many assignments need to be done, they will then be awarded the pre-determined reward to play in the computer!
